## ⚠️ 诚实边界

1. **公开表达 ≠ 全部真相** — 老王在直播间讲的是"他选择讲的故事"
2. **回顾偏差** — 从2023年回顾2000年代资本运作，叙事必然经过事后修正
3. **外部评价冲突** — 媒体称其"资本掮客" vs 自我定位"中国式PE"
4. **玄学叙事争议** — "邵子神数"等经历无法被第三方验证
5. **时效性** — 近年主业是自媒体，投资观点属"过来人经验"而非"当前判断"
6. **粉丝滤镜** — 衍生频道内容经过二次筛选加工
